# Calculator Project

## Description
This is a simple calculator application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the project in browser
2. Enter numbers
3. Click buttons to perform calculations
